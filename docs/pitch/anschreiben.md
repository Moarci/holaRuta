### Die fünf Fragen für ein Feedback-Gespräch (jede Zielgruppe)
1. Würden eure Leute so etwas vor einer LatAm-Reise/zum Kursstart nutzen?
2. Könnte eine Lehrkraft / ein Koordinator damit eine kurze Gruppen-Aktivität machen?
3. Welche Situationen fehlen aus eurer Sicht?
4. Nettes Extra oder echter Mehrwert?
5. **Wen müsste man bei euch ansprechen, um einen Pilot zu testen?** ← die wichtigste.
