🧑‍🏫 Vorschau · Ziel-Picker

Interaktive Vorschau der neuen Ziel-Auswahl im Lehrer-/Reiseleiter-Modus. Tippe auf das Ziel-Feld → das Modal mit den drei erklärten Gruppen öffnet sich. Wähle ein Ziel, das Feld aktualisiert sich. (Vorher: nüchterne System-Dropdown-Liste.)

Aufgabe erstellen & teilen

Wähle ein Ziel, erzeuge einen Code und teile ihn als Link. Ein Antippen abonniert die Aufgabe bei den Lernenden.

🎯 Aufgaben-Ziel wählen

Worauf soll sich die Aufgabe beziehen? Wähle ein Ziel – die Lernenden bekommen genau diese Karten. Jede Gruppe unten erklärt, wann sie passt.
