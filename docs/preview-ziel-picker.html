🧑‍🏫 Vorschau · Ziel-Picker & Bundles

Interaktive Vorschau des Modo profe. Tippe aufs Ziel-Feld → das Modal öffnet sich. Oben fertige Bundles (Vorlage übernehmen = Auswahl als Startpunkt setzen), darunter die drei erklärten Gruppen als Mehrfachauswahl. 1 Ziel = Einzelaufgabe, ≥2 Ziele = Bundle (ein Link, mehrere Aufgaben). Schließen via ✕ / Fertig / Scrim / Escape.

Aufgabe / Bundle erstellen & teilen

Wähle ein Ziel oder stell ein Bundle zusammen, erzeuge einen Code und teile ihn als Link.

🎯 Aufgaben-Ziel wählen

Worauf soll sich die Aufgabe beziehen? Wähle ein Ziel – die Lernenden bekommen genau diese Karten. Tippe mehrere Ziele an, um daraus ein Bundle zu bauen – ein Link, der mehrere Aufgaben auf einmal anlegt.
